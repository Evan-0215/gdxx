# 玩具厂订单跟进系统 - 规格说明 v2

## 1. Concept & Vision

专为玩具厂老板设计的精细化订单追踪工具。采用任务看板风格，每个订单的每个环节（报价、生产、物料等）都有独立的任务清单，确保50+订单的每个细节都不遗漏。界面清晰高效，支持本地存储。

## 2. 设计结构

### 订单基础信息
- 订单编号（自动生成）
- 客户名称
- 产品名称
- 产品数量
- 下单日期
- 交货日期
- 备注

### 订单待办大项（每个大项包含小项清单）

| 大项 | 小项 |
|------|------|
| **报价办** | 设计文稿介绍会 → 材料选择 → 材料/工艺清单及完成设计 → 3D拆图及打印 → 布料回厂 → 首轮样板 → 客人comment及改善 → 报价 |
| **生产办** | 打样申请 → 样品确认 → 大货审批 → 生产许可证 |
| **放产资料** | BOM表整理 → 工艺图纸 → 作业指导书 → 发料单 |
| **物料请购** | 物料清单核对 → 供应商询价 → 采购下单 → 物料到货确认 |
| **加工商确认** | 加工商筛选 → 合同签订 → 技术交底 → 交期确认 |
| **加工商生产跟进** | 首件确认 → 进度跟进 → 品质检验 → 出货安排 |
| **内铺生产安排** | 排产计划 → 产线分配 → 生产日报 → 完成入库 |

## 3. Design Language

**色彩方案**:
- 主色: `#6366F1` (靛蓝 - 专业)
- 背景: `#0F172A` (深色)
- 卡片: `#1E293B`
- 文字: `#F8FAFC`
- 状态色:
  - 待办: `#64748B` (灰)
  - 进行中: `#F59E0B` (橙)
  - 已完成: `#10B981` (绿)
  - 逾期: `#EF4444` (红)

## 4. 功能

1. **订单列表** - 卡片展示订单基本信息
2. **订单详情面板** - 右侧滑出，显示所有大项和小项
3. **任务清单** - 每个大项展开显示小项，可勾选完成
4. **进度计算** - 自动计算每个大项和总体进度
5. **逾期提醒** - 标记逾期任务
6. **搜索筛选** - 按客户/产品/交货日期筛选
7. **数据导入导出**

## 5. Technical

- 单 HTML 文件，内嵌 CSS/JS
- localStorage 持久化
- 响应式布局
